# 3. GUIA DE EDICAO DE HIGHLIGHTS

## 3.1 Estrutura do Highlight Perfeito

### Duracao Ideal por Plataforma
```
TikTok/Reels:     15-30 segundos (maximo 60s)
YouTube Shorts:   30-45 segundos
YouTube normal:   2-5 minutos (compilacao)
Stories:          15 segundos por story
Twitter/X:        15-45 segundos
```

### Anatomia do Highlight

```
ESTRUTURA 30 SEGUNDOS:

[0-3s]   HOOK - Momento mais impactante (spoiler do final)
[3-5s]   TRANSICAO - Volta para o contexto
[5-20s]  BUILD UP - Construcao da jogada
[20-28s] CLIMAX - A jogada acontecendo
[28-30s] REACAO - Comemoração/reacao (se tiver)

IMPORTANTE:
- NUNCA comece devagar
- Os 3 primeiros segundos definem se a pessoa fica
- O momento HYPE deve aparecer JA no inicio (teaser)
```

## 3.2 Cortes e Timing

### Ritmo de Cortes
```
Momento normal:     Corte a cada 2-3 segundos
Momento de tensao:  Cortes mais longos (4-5s) para construir
Momento HYPE:       Cortes rapidos (0.5-1s) no climax
Pos-jogada:         Slow motion + corte normal
```

### Tecnicas de Corte

**Jump Cut**
- Usar entre rounds ou momentos "mortos"
- Bom para acelerar setup de jogada

**Match Cut**
- Cortar de um tiro para outro
- Manter fluidez entre kills

**Smash Cut**
- Corte abrupto para momento de impacto
- Ideal para clutches

**Slow Motion**
- 50% velocidade para headshot importante
- 25% velocidade para o kill final do ace
- Nunca mais que 3 segundos em slow

## 3.3 Transicoes Recomendadas

### Transicoes que Funcionam em E-sports

```
CORTE SECO (mais usado - 60% do video)
- Profissional, rapido, nao distrai
- Usar na maioria dos momentos

ZOOM IN/OUT (20%)
- Zoom in rapido no momento do kill
- Zoom out para mostrar contexto

GLITCH/DISTORCAO (10%)
- Para momentos "WTF" ou insanos
- Nao abusar - max 2x por video

WHIP PAN (5%)
- Transicao rapida lateral
- Entre clips diferentes

FADE ROXO (5%)
- Para finalizar videos
- Para separar "capitulos"
```

### Transicoes para EVITAR
```
- Star wipe ou qualquer wipe "infantil"
- Transicoes 3D elaboradas
- Dissolve lento (mata o ritmo)
- Flip de pagina
- Qualquer coisa que pareca PowerPoint anos 2000
```

## 3.4 Texto e Graficos no Video

### Texto On-Screen

```
ESTILO:
- Fonte: BEBAS NEUE ou similar (bold, sem serifa)
- Cor principal: Branco com borda preta
- Cor destaque: Roxo neon ou verde (para kills)
- Tamanho: Grande o suficiente para mobile
- Posicao: Terco inferior ou superior (nunca no meio)

QUANDO USAR:
- Nome do player no primeiro momento
- "4K", "ACE", "CLUTCH" quando acontece
- Placar importante
- Contexto rapido ("1v3", "match point")

ANIMACAO:
- Entrada rapida (pop ou slide)
- Duracao: 1-2 segundos max
- Saida: fade ou pop out
```

### Kill Counter
```
Mostrar contador de kills durante ace/multi-kill:
  1K -> 2K -> 3K -> 4K -> ACE

- Cada numero aparece no momento do kill
- Som de "ding" ou efeito sincronizado
- Cor muda: branco -> amarelo -> laranja -> vermelho -> ROXO NEON
```

## 3.5 Audio e Musica

### Estrutura de Audio

```
CAMADAS:
1. Audio do jogo (30-50% volume)
2. Musica de fundo (60-70% volume)
3. Efeitos sonoros extras (80-100% volume)
4. Reacao/comms (quando tiver) (100% volume)

MIXAGEM:
- Baixar musica nos momentos de fala/reacao
- Subir audio do jogo no momento do tiro
- Musica deve complementar, nao competir
```

### Musicas que Funcionam

```
GENEROS:
- Phonk (o mais usado em CS2 highlights)
- Trap beats
- Drum and Bass
- EDM/Future Bass para momentos epicos
- Lo-fi para momentos mais chill

ONDE ENCONTRAR (royalty free):
- Epidemic Sound (pago, mas top)
- Artlist (pago)
- NCS (gratis, dar creditos)
- Pixabay Music (gratis)
- YouTube Audio Library (gratis)
- Streambeats (gratis, feito para streamers)

BPM RECOMENDADO:
- Highlights normais: 120-140 BPM
- Momentos HYPE: 140-160 BPM
- Compilacao chill: 80-100 BPM
```

### Efeitos Sonoros

```
ESSENCIAIS:
- Boom/impact no momento do kill
- Whoosh nas transicoes
- Riser antes do momento climax
- Reverse cymbal para build up

ONDE BAIXAR:
- Freesound.org (gratis)
- Mixkit (gratis)
- Zapsplat (gratis com cadastro)
```

## 3.6 Color Grading para Highlights

```
LUT BASE "ORBITAL GAMING":

Lift (Sombras):   Adicionar um toque de azul escuro
Gamma (Medios):   Levemente dessaturar, puxar roxo
Gain (Realces):   Manter neutro ou leve cyan

Curvas:
- Levantar pretos levemente (nao ter preto puro)
- Comprimir realces (evitar branco estourado)

Saturacao seletiva:
- Aumentar vermelhos (sangue, kills)
- Aumentar roxos/magentas (identidade)
- Diminuir amarelos/laranjas (menos "quente")
```

# 4. STORYTELLING VISUAL CUP #1

## 4.1 Arco Narrativo do Evento

### Estrutura em 3 Atos

```
ATO 1 - PREPARACAO (Pre-evento)
"A jornada comeca"
- Expectativa e tensao
- Conhecendo os protagonistas
- Definindo o desafio

ATO 2 - BATALHA (Durante evento)
"O confronto"
- Altos e baixos
- Momentos de duvida
- Viradas dramaticas

ATO 3 - GLORIA/APRENDIZADO (Pos-evento)
"A conclusao"
- Resultado e emocoes
- Reflexao sobre a jornada
- Setup para o proximo capitulo
```

## 4.2 Timeline de Conteudo Visual

### SEMANA -1 (Pre-Evento)

```
DIA -7 (Segunda):
POST: Anuncio oficial com arte elaborada
STORY: Countdown comeca (7 dias)
FORMATO: Carrossel com todas as infos

DIA -5 (Quarta):
POST: Apresentacao dos times participantes
REELS: Teaser trailer 15-30s com musica epica
STORY: Enquete "Quem vai ganhar?"

DIA -3 (Sexta):
POST: Chave de confrontos visual
STORY: "Conheca os players" (um por story)
REELS: Highlights de cada time (se tiver)

DIA -1 (Domingo):
POST: "AMANHA" - arte de hype maximo
STORY: Bastidores da preparacao
LIVE: Q&A com players (opcional)
```

### DIA DO EVENTO

```
MANHA:
STORY: "Hoje e o dia" + cronograma visual
POST: Lembrete com horarios

DURANTE AS PARTIDAS:
STORY: Atualizacao de placar a cada partida
STORY: Screenshots de momentos epicos
STORY: Reacoes dos players (se tiver)

ENTRE PARTIDAS:
STORY: Resumo rapido da partida anterior
STORY: Proxima partida em X minutos

FINAL:
STORY: Momentos finais com tensao maxima
POST: Resultado com foto/arte do campeao
REELS: Highlight do momento campeonato
```

### SEMANA +1 (Pos-Evento)

```
DIA +1:
POST: Foto oficial do campeao
REELS: Melhores momentos compilacao (60s)
STORY: Agradecimento + bastidores

DIA +2:
POST: MVPs e estatisticas do campeonato
CARROSSEL: Top 5 plays com screenshots

DIA +3:
REELS: Highlight individual do MVP
POST: Quote marcante do evento

DIA +5:
POST: "Relembre o Cup #1" - carrossel nostalgia
REELS: Mini documentario (2-3 min) - para YouTube tambem

DIA +7:
POST: "E o proximo?" - teaser Cup #2
STORY: Enquete sobre melhorias/sugestoes
```

## 4.3 Tipos de Conteudo por Momento

### Conteudo de HYPE (Pre-evento)

```
FORMATOS:
1. Trailer cinematico (30-60s)
   - Musica epica, cortes rapidos
   - Logos dos times, nomes dos players
   - Data e horario em destaque

2. Countdown posts
   - "3 DIAS" com visual impactante
   - Stories diarios com contagem

3. Versus cards
   - Time A vs Time B
   - Lado a lado, visual de confronto

4. Player spotlights
   - Cada player com stats/historico
   - Estilo "trading card"
```

### Conteudo LIVE (Durante evento)

```
FORMATOS:
1. Placar ao vivo (Stories)
   - Atualizar a cada round importante
   - Template pre-definido, so trocar numeros

2. Momentos instantaneos
   - Screenshot rapido + texto
   - "CLUTCH DO [PLAYER]!"

3. Mini-clips (15s max)
   - Jogada que acabou de acontecer
   - Postar em menos de 5 minutos

4. Reacoes/emocoes
   - Fotos de celular dos players
   - Momentos genuinos
```

### Conteudo REFLEXAO (Pos-evento)

```
FORMATOS:
1. Compilacao de melhores momentos
   - 60s para Reels, 3-5min para YouTube

2. Estatisticas visuais
   - Infograficos com dados do campeonato
   - MVP, mais kills, melhor clutch etc

3. Depoimentos (se tiver)
   - Players falando sobre o evento
   - Formato talking head ou texto

4. Behind the scenes
   - Bastidores, setup, preparacao
   - Humaniza a marca

5. Mini documentario
   - Contar a historia completa
   - 3-5 minutos, para YouTube
```

## 4.4 Narrativa Visual: Elementos de Historia

### Protagonistas
```
- Time campeao como heroi principal
- MVP como destaque individual
- Underdogs como arco de superacao
- Rivalidades para criar tensao
```

### Conflito
```
- Partidas equilibradas
- Viradas improvaveis
- Match points salvos
- Clutches decisivos
```

### Emocao
```
- Tensao nos momentos decisivos
- Alegria na vitoria
- Frustracao na derrota (com respeito)
- Camaradagem entre players
```

### Visual Storytelling
```
ANTES:     Cores mais frias, tensao
DURANTE:   Contraste alto, energia
CLIMAX:    Saturacao maxima, efeitos
DEPOIS:    Tons mais quentes, satisfacao
```

# 5. CONTEUDO REELS/TIKTOK

## 5.1 Formatos que Funcionam em E-sports

### FORMATO 1: O Highlight Puro (Mais engajamento)

```
ESTRUTURA:
[0-1s]   Kill/momento insano (hook)
[1-3s]   Texto: "esse clutch..."
[3-25s]  A jogada completa
[25-30s] Reacao ou slow-mo do final

AUDIO:
- Som trending ou phonk beat
- Audio do jogo nos momentos chave

EXEMPLO:
"Ele salvou o campeonato com esse clutch"
+ Jogada completa
+ Slow motion no ultimo kill
```

### FORMATO 2: POV/Primeira Pessoa

```
ESTRUTURA:
[0-2s]   Texto: "POV: voce e o ultimo vivo"
[2-25s]  Gameplay em primeira pessoa
[25-30s] Momento epico + reacao

ESTILO:
- Imersivo, como se fosse o viewer jogando
- Funciona bem com audios de tensao
- Texto em segunda pessoa ("voce")
```

### FORMATO 3: Before/After (Transformacao)

```
ESTRUTURA:
[0-3s]   "Antes: perdendo de 14x8"
[3-5s]   Clip triste/perdendo
[5-8s]   Transicao dramatica
[8-25s]  "Depois: reverse sweep"
[25-30s] Comemoracao/resultado final

AUDIO:
- Musica triste -> transicao -> musica hype
- Ou audio trending que tenha essa estrutura
```

### FORMATO 4: Compilacao Rapida

```
ESTRUTURA:
Cortes de 1-2 segundos cada
4-8 clips em sequencia
Sincronizado com batida da musica

EXEMPLO:
"5 aces em um campeonato"
[Kill] [Kill] [Kill] [Kill] [Kill]
Cada kill no beat da musica

AUDIO:
- Musica com batida forte e marcada
- Phonk funciona perfeitamente
```

### FORMATO 5: Tutorial/Dica Rapida

```
ESTRUTURA:
[0-2s]   "Como fazer [jogada/smoke/flash]"
[2-20s]  Demonstracao passo a passo
[20-30s] Resultado + "salva pra depois"

ESTILO:
- Texto explicativo durante
- Pode usar voz (AI ou real)
- CTA no final: "salva e usa no proximo jogo"
```

### FORMATO 6: Storytelling/Mini Doc

```
ESTRUTURA:
[0-5s]   Hook: resultado final ou momento epico
[5-15s]  Contexto: como chegou ali
[15-50s] Historia com texto/narracao
[50-60s] Conclusao emocional

AUDIO:
- Musica cinematica
- Pode ter narracao
- Audio ambiente do jogo

IDEAL PARA:
- Historias de underdogs
- Jornada de um player
- Resumo de campeonato
```

### FORMATO 7: Trend/Meme Adaptado

```
ESTRUTURA:
- Pegar audio/formato trending
- Adaptar para contexto de CS2
- Manter timing do original

EXEMPLOS:
- "POV" trends
- Audio de filmes/series
- Challenges adaptados
- Memes do momento

ONDE ACHAR TRENDS:
- Aba Reels do Instagram (ver o que ta bombando)
- TikTok For You
- Twitter gaming
- Competidores (Santos Games, etc)
```

### FORMATO 8: Reacao/Commentary

```
ESTRUTURA:
[0-2s]   Rosto reagindo (thumbnail)
[2-25s]  Gameplay + reacao em PiP
[25-30s] Reacao final exagerada

ESTILO:
- Picture-in-picture (rosto + gameplay)
- Funciona se tiver personalidades
- Reacoes genuinas > reacoes forcadas
```

## 5.2 Hooks que Funcionam

### Hooks de Texto (Primeiros 2 segundos)

```
CURIOSIDADE:

"Isso nao era pra dar certo, mas..."

DESAFIO:
"Tenta fazer isso no seu jogo"

"99% erra esse smoke"

NUMERO/ESTATISTICA:
"1v5 clutch em menos de 30 segundos"
"5 headshots consecutivos"

EMOCAO:
"O round que mudou tudo"
"A jogada que salvou o campeonato"

POV:
"POV: ultimo vivo, bomb plantada"

CONTROVERSIA (com cuidado):

```

## 5.3 Especificacoes Tecnicas

```
DIMENSOES:
- Vertical: 1080x1920 (9:16)
- Feed quadrado: 1080x1080 (se necessario)

DURACAO IDEAL:
- Hook: 1-3 segundos
- Total TikTok: 15-30 segundos
- Total Reels: 15-60 segundos
- Maximo engajamento: 7-15 segundos

SAFE ZONES:
- Topo: 150px (username/titulo)
- Baixo: 270px (botoes)
- Texto importante no centro

TAXA DE FRAMES:
- 60fps preferivel
- 30fps minimo aceitavel
```

# 6. COMO APROVEITAR MATERIAL NAO-PROFISSIONAL

## 6.1 Fotografia de Celular com Qualidade

### Antes de Fotografar

```
ILUMINACAO:
- LUZ NATURAL e sua melhor amiga
- Fotografar perto de janelas
- Evitar luz direta do sol (sombras duras)
- A noite: usar luz de ring light ou monitor

ANGULO:
- Nivel dos olhos ou levemente acima
- Evitar angulo de baixo (queixo duplo)
- Fotos de setup: angulo de 45 graus

FUNDO:
- Limpar a bagunca do fundo
- Setup arrumado = foto boa
- Luzes RGB no fundo ajudam MUITO

CAMERA:
- Limpar a lente do celular (sempre suja)
- Usar camera traseira (melhor qualidade)
- Evitar zoom digital (perde qualidade)
- HDR ligado para interiores
```

### Configuracoes do Celular

```
IPHONE:
- Modo Retrato para pessoas
- ProRAW se disponivel (mais edicao)
- Grid ligado (composicao)
- HDR automatico

ANDROID:
- Modo Pro/Manual se tiver
- 12MP ou resolucao nativa
- Desligar filtros automaticos
- Salvar em JPEG alta qualidade
```

### Tipos de Fotos Uteis

```
1. SETUP DO PLAYER
   - Teclado + mouse + monitor
   - Iluminacao RGB ajuda muito
   - Angulo de 45 graus diagonal

2. REACAO/EMOCAO
   - Capturar momento genuino
   - Comemoracao, tensao, frustracao
   - Nao precisa estar "posada"

3. BASTIDORES
   - Preparacao antes do jogo
   - Conversas em grupo
   - Momentos descontraidos

4. DETALHES
   - Mao no mouse durante jogada
   - Tela com o jogo
   - Periphericos e equipamentos

5. GRUPO/TIME
   - Todos juntos (mesmo online, print)
   - Uniforme/camiseta se tiver
   - Logo aparecendo
```

## 6.2 Melhorando Fotos Ruins na Edicao

### Problemas Comuns e Solucoes

```
PROBLEMA: Foto escura
SOLUCAO:
- Aumentar exposicao +30 a +50
- Levantar sombras +50 a +70
- Diminuir realces -20
- Adicionar contraste +15

PROBLEMA: Foto amarelada (luz artificial)
SOLUCAO:
- Diminuir temperatura de cor (-15 a -30)
- Ou converter para P&B com toque de roxo

PROBLEMA: Foto granulada (ISO alto)
SOLUCAO:
- Reducao de ruido (com moderacao)
- Adicionar granulado estilizado (disfarcar)
- Diminuir claridade

PROBLEMA: Fundo bagunçado
SOLUCAO:
- Desfocar fundo (blur)
- Escurecer bordas (vinheta pesada)
- Cropar eliminando areas ruins

PROBLEMA: Pessoa borrada
SOLUCAO:
- Se leve: aumentar claridade + nitidez
- Se pesado: converter para "efeito artistico"
- Usar como background desfocado
```

### Filtros que Salvam Fotos Ruins

```
PRETO E BRANCO COM ROXO (Duotone):
1. Converter para P&B
2. Adicionar overlay roxo em "Soft Light"
3. Aumentar contraste
= Qualquer foto fica "artistica"

HIGH CONTRAST:
1. Contraste +40
2. Sombras -30
3. Realces -40
4. Claridade +20
= Esconde imperfeicoes

GRAIN VINTAGE:
1. Dessaturar -20
2. Granulado +40
3. Fade pretos +15
4. Vinheta +30
= Visual "proposital" lo-fi
```
