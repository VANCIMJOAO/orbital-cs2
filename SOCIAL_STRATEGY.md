# 4. FORMATOS DE POSTS RECOMENDADOS

## REELS (Principal formato - 60% dos posts)

### Especificacoes Tecnicas
- **Duracao ideal:** 15-30s para highlights, 45-60s para tutoriais
- **Aspecto:** 9:16 (vertical)
- **Resolucao:** 1080x1920
- **Primeiros 3 segundos:** HOOK forte (texto impactante ou jogada insana)

### Estrutura de Reels de Highlight
```
0-3s: Hook visual (momento mais insano)
3-7s: Build up / contexto
7-12s: Climax da jogada
12-15s: Reacao + CTA texto
```

### Tipos de Reels por Performance
1. **Clutch isolado** (15s) - Maior potencial viral
2. **Compilacao top 5** (30-45s) - Alto watch time
3. **POV gameplay** (20-30s) - Imersivo
4. **Reacao da crowd** (15s) - Emocional
5. **Tutorial rapido** (30-45s) - Valor agregado

### Elementos de Edicao
- Texto grande e legivel (fonte bold, contraste alto)
- Zoom nos momentos cruciais
- Slow motion no clutch
- Audio synced com a jogada
- Logo ORBITAL no canto

---

## CARROSSEL (25% dos posts)

### Especificacoes Tecnicas
- **Slides:** 5-10 (ideal 7)
- **Aspecto:** 1:1 ou 4:5
- **Resolucao:** 1080x1080 ou 1080x1350

### Estrutura de Carrossel Efetivo
```
Slide 1: Hook forte (titulo impactante)
Slide 2-6: Conteudo principal
Slide 7: CTA (salva, comenta, segue)
```

### Tipos de Carrossel
1. **Educativo:** "5 dicas para melhorar no CS2"
2. **Recap:** "Tudo que rolou na Cup #1"
3. **Ranking:** "Top 5 jogadas do mes"
4. **Behind the scenes:** "Como organizamos a Cup"
5. **Produto:** "Nova colecao ORBITAL"

### Design Guidelines
- Identidade visual consistente (roxo, preto, neon)
- Setas indicando "deslize"
- Numeracao dos slides
- CTA no ultimo slide sempre

# 8. ESTRATEGIA DE ENGAJAMENTO COM COMUNIDADE

## Resposta a Comentarios

### Tempo de Resposta
- **Primeira hora:** Responder TODOS os comentarios
- **Primeiras 24h:** Monitorar e responder novos
- **Objetivo:** 100% de taxa de resposta

### Tom de Voz nas Respostas
```
FAZER:
- Usar nome/@ da pessoa
- Ser informal mas respeitoso
- Fazer perguntas de volta
- Usar girias da comunidade gamer
- Reconhecer contribuicoes

NAO FAZER:
- Respostas genericas (tipo "obrigado!")
- Ignorar criticas construtivas
- Ser formal demais
- Demorar mais de 24h pra responder
```

### Exemplos de Respostas
```
Comentario: "Esse clutch foi insano"
Resposta: "@user mano esse cara eh outro nivel, tu tava la no dia?"
```

## Gestao de Crise

### Tipos de Situacao e Resposta

**Comentario negativo publico:**
```
1. Nao deletar (a menos que seja ofensivo)
2. Responder publicamente de forma madura
3. Oferecer resolver por DM
4. Fazer follow-up
```

**Fake news ou informacao errada:**
```
1. Corrigir educadamente com fatos
2. Postar esclarecimento oficial se necessario
3. Fixar comentario com a informacao correta
```

**Trolls:**
```
1. Ignorar provocacoes sem fundamento
2. Nao alimentar discussoes improdutivas
3. Bloquear em casos extremos
4. Manter a compostura sempre
```
